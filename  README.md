# Lottery

## Libraries
- yarn v1.6.0
- node v14.15.4
- react v17.0.1
- redux v4.0.5
- next v10.0.8
- styled-components v5.2.1
- @material-ui v4.11.3

## Install Dependencies
```
yarn install
```
## Get Started
Development
```
yarn dev
```
Start
```
yarn build
yarn start
```
